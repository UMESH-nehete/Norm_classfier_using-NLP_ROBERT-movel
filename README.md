# Norm_classfier_using-NLP_ROBERT-movel
Problem outline: A norm is a behavioral expectation, practice, or value that members of a culture follow. A norm must describe WHAT PEOPLE DO or SHOULD DO. For example, in India, touching feet of elders is a norm. In America, tipping in restaurant is a norm. In Japan, however, tipping is often not expected. On the other hand, a generic sentence describes factual information, observations, or general statements without prescribing behavior. The task in the project is to develop a model to distinguish between a generic sentence and a cultural norm.
